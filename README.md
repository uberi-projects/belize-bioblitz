# belize-bioblitz <img src='www/images/UBERI_Logo.png' align="right" height="70" />

![status](https://img.shields.io/badge/status-continuous%20dev.-blue)

This repository is intended to store the data and scripts resulting from Belize Bioblitz efforts conducted by the Univeristy of Belize Environmental Research Institute (UB-ERI) and partners. 
Each folder pertains to a specific bioblitz, and all resources for that bioblitz may be found within that folder. To learn more about the UB-ERI, navigate to https://www.uberibz.org/.
